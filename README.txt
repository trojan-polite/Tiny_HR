== Installation ==
1. Download the zip file, and unzip it.
1. Upload the entire folder (simple-fading-testimonials) to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress


== Description ==

The Simple Fading Testimonials Widget enables easy to add revolving and fading testimonials through the widget area. 


== Installation ==

1. Upload `Simple-Fading-Testimonials` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to "Appearance" -> "Widgets" and add a Simple Fading Testimonials widget to a sidebar of your choosing.



== Frequently Asked Questions ==

= Can I adjust the number of testimonials to display? =

Yes, all settings are controlled via the "Appearance" -> "Widgets" area.
